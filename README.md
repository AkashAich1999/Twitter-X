# Twitter-X

A full-stack MERN-based Twitter-X with authentication, posting, following, and more.

## Features
- User Signup/Login
- JWT Authentication
- Post Tweets with Images
- Follow/Unfollow Users
- Like and Comment on Posts

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Cloudinary (for image upload)
